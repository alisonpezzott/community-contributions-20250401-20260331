## About Me

I am a **Microsoft MVP in Data Platform** — an analytics engineering specialist dedicated to building modern, scalable, and collaborative data solutions on the Microsoft stack. My work sits at the intersection of **data engineering**, **business intelligence**, and **DevOps**, helping individuals and organizations move beyond static dashboards toward fully automated, production-grade data platforms.

I am the author of **[pyFabricOps](https://pypi.org/project/pyfabricops/)**, an open-source Python library that provides a clean, consistent interface to the Microsoft Fabric and Power BI REST APIs — featured in [The Fabric Essentials Listings](https://www.linkedin.com/posts/welcome-to-the-fabric-essentials-listings-share-7446857843967725568-w1b2) and used by data professionals and DevOps engineers worldwide.

As a **community educator**, I produce original technical content on YouTube, write in-depth articles on my blog, and speak at major industry events including **MVP Conf**, **TDC**, **SQL Saturday**, **Power BI Meeting Brazil**, and **Microsoft Reactor**. I am passionate about closing the gap between data engineering and BI development — and making CI/CD a first-class citizen in every data project.

### 💻 [Open Source Code & Projects](CODE.md)

GitHub repositories and community code contributions built to solve real-world data engineering and DevOps challenges.

**Flagship project — pyFabricOps:**
A Python wrapper library for Microsoft Fabric and Power BI REST APIs, designed to run in Fabric notebooks, pure Python scripts, or as part of YAML-based CI/CD workflows. Available on [PyPI](https://pypi.org/project/pyfabricops/) with full [documentation](https://pyfabricops.readthedocs.io/) and featured in **The Fabric Essentials Listings**.

**Other highlights:**
- **FabricOpsFlow** – End-to-end CI/CD framework for Microsoft Fabric with Azure DevOps Pipelines
- **FabricOpsFlow-PBI-Pro** – CI/CD framework tailored for Power BI PRO projects
- **calendario-sql** – Calendar table generator compatible with SQL Server, Azure SQL, and Microsoft Fabric
- **pbi-ci-cd-isv-multi-tenant** – Multi-tenant CI/CD scenario for Power BI PRO with GitHub Actions
